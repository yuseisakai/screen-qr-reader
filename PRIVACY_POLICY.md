# Privacy Policy for 画面QRリーダー

Last updated: June 7, 2026

画面QRリーダーは、表示中のページにあるQRコードを読み取り、ユーザーが確認したURLを開くためのChrome拡張機能です。

## Data Collection

This extension does not collect, store, sell, or share personal information.

When the user clicks the scan button, the extension temporarily captures the visible area of the active browser tab to detect QR codes. This screenshot is processed locally in the browser and is not sent to any external server.

## Data Use

The captured image is used only to detect and decode QR codes on the visible page. The decoded result is shown in the extension popup.

If the decoded result is an `http:` or `https:` URL, the user can choose to open it in a new tab. Non-URL QR code contents are displayed only and are not opened automatically.

## Data Sharing

This extension does not share user data with any third parties.

## Remote Code, Analytics, and Tracking

This extension does not use remote code, external APIs, analytics tools, advertising SDKs, or tracking technologies.

## Permissions

The extension uses the `activeTab` permission only when the user interacts with the extension. This permission is used to capture the currently visible tab area for QR code detection.

## Contact

For questions about this privacy policy, contact the developer through the Chrome Web Store listing or the project repository.
